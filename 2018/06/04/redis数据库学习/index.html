redis入门

redis简介

  1. Redis支持数据持久化，可以将内存中的数据保存在磁盘中，重启的时候再次进行加载使用。–读写速度快，操作简单
  2. Redis支持数据的备份，即master-slave模式的数据备份
  3. Redis不仅仅支持简单的key-value类型的数据，同时还提供list，set，zset，hash等数据结构的存储。–丰富的数据类型
  4. Redis的所有操作都是原子性的，意思就是要么成功执行要么失败完全不执行。单个操作是原子性的。多个操作也支持事务，即原子性，通过MULTI和EXEC指令包起来。

redis数据类型

  1. string
  2. hash

redis命令

  1. redis客户端基本语法redis-cli。执行 PING 命令，该命令用于检测 redis 服务是否启动。
