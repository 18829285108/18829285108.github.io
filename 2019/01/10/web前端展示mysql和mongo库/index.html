web前端展示mysql和mongo库

Posted on 2019-01-10

目的

将mysql数据库或者mongo数据库，放到网页上面，随时可以添加不同数据库ip，页面操作更简单

phpmyadmin–mysql数据库

如果没有phpmyadmin镜像，需要下载一个。docker pull phpmyadmin/phpmyadmin

新建phpMyAdmin文件夹。编辑config.inc.php文件

rockmongo–mongo数据库

参考链接：https://blog.csdn.net/boling_cavalry/article/details/78234762 如果没有rockmongo镜像，需要下载一个。docker pull javierjeronimo/rockmongo:latest

新建rockmongo文件夹，编辑config.php文件
